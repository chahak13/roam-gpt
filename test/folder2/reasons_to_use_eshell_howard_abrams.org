#+title: Reasons to use EShell - Howard Abrams

+ EShell is a shell, not a terminal.


1. It's an elisp interpreter
2. Can mix lisp expressions and shell expressions
3. Basically use eshell as an emacs buffer that /can/ do shell things and not as a shell buffer
