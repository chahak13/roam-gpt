#+TITLE: Change font size in emacs
#+DATE: 2021-06-25
#+filetags: emacs

Text in emacs can be scaled using '=C-x C-+=' or '=C-x C-=' to increase and '=C-x C--=' to decrease. =C-x C-0= can be used to bring it back to the default size.
