                 ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━


Matplotlib has a `matplotlibrc' file that sets up many different values
that in one way or other affect the usage of the library. This note
tries to document which `rcParams' value is used where in matplotlib.
While the comments are useful, they don’t always list the secondary
effects of setting some values (ex. `linewidth'). This should hopefully
address that too.
